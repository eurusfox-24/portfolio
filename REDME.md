# My Professional Portfolio

This is a lightweight, terminal-themed portfolio hosted on GitHub Pages.

## How to Edit
- **Text:** Open `index.html` and change the text inside the `<h3>` and `<p>` tags.
- **Images:** Replace `project-pic1.jpg` in the folder with your own photos.
- **CV:** Name your CV file `your-cv.pdf` and place it in the main folder.

## Sections
1. **Projects:** Focuses on AI and Middleware integration.
2. **Homelab:** Showcases Docker and Infrastructure setups.
